**Installing**
``` sh
$ git clone https://github.com/black-software-com/Black-Webbrowser

$ cd Black-Webbrowser

$ bash install.sh
```
<br>

*Argument*
``` txt
Black-Webbrowser Argument:
        python black 
        or
        python black <URL>
        
        python black --version | version
        
        python black --help | help
```
<br>

**Update**
``` sh
$ cd /Update

$ python update.py

```

**Uninstalling**
``` sh
$ cd Uninstall

$ python uninstall.py
```
